## 🚀 Getting Started

To run this portfolio locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/visusarvaiya/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run in development mode:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
